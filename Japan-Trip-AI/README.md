# Japan Trip AI Assistant

## What It Does
No-code Agentic AI travel planning workflow 
that processes any destination query and 
delivers personalised trip recommendations 
via email automatically.

## Features
Dynamic prompt engineering with 13 
personalisation parameters including:
Budget limit — Food preferences — 
Walking limit — Peaceful locations — 
No seafood — First time visitor context

## Results
Full Japan trip planned within 2000 euros
Peaceful hidden spots suggested
Chicken and egg food only respected
5 day itinerary delivered by email

## Tools Used
Make.com | Make AI Toolkit | Gmail

## Workflow
[Webhook] → [AI] → [Extract Info] → [Gmail]

## Key Achievement
AI understood user INTENT not just 
instructions — suggested secret villages 
with almost no tourists without being asked!
